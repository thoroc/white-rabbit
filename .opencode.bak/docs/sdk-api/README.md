# Generated SDK API docs

This directory is generated by `scripts/extract-sdk-docs.ts` and contains per-package API documentation.

Files:

- 'INDEX.md' - human-readable index of scanned packages
- 'INDEX.json' - machine-readable index (packages, counts, anchors)
- per-package directories with `API.md` and per-file docs

Regenerate by running:

    npm run docs:api	# or: bun ./scripts/extract-sdk-docs.ts
