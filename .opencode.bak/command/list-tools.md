# List Available Tools

Please provide a comprehensive list of all tools you have access to in this session.

## Requirements

For each tool, provide:

1. **Exact tool name** (as it would be used in function calls)
2. **Brief description** of what the tool does
3. **Category** (if applicable: file operations, shell, web, etc.)

## Output Format

Present the tools in a clear, organized format such as:

### File Operations

- `read` - Read file contents
- `write` - Write file contents
- `edit` - Edit file with string replacement

### Shell Operations

- `bash` - Execute bash commands

### Web Operations

- `webfetch` - Fetch web content

(etc.)

## Instructions

- List ALL available tools, not just commonly used ones
- Use the exact names as they appear in your tool definitions
- Group by category for better readability
- Include any specialized tools or plugins currently loaded
